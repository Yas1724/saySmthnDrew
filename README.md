# SaySmthnDrew

A visually expressive web page that combines a simple text prompt with a beautiful, animated floral background. This project creates a serene and aesthetic digital space, perfect for a personal message or artistic display.

## ✨ Features

*   **Animated Floral Background**: Features a high-quality, looping video of blooming watercolor flowers as the full-screen background.
*   **Clean Text Overlay**: A simple, centered text prompt ("Say smthn drew") invites interaction or reflection.
*   **Aesthetic Design**: Combines soft, artistic visuals with a minimalist text interface for a calming user experience.
*   **Responsive Layout**: The text overlay is centered and scales appropriately on different screen sizes.

## 🛠️ Technologies Used

*   **HTML5**: Provides the structure of the page, including the video background element.
*   **CSS3**: Styles the page layout, centers the text, and ensures the video covers the background correctly.
*   **Visual Assets**: Uses a `.jpg` video file (`watercolor-beautiful-floral-bloom-animation-of-colorful-flowers-banner-animated-4k-blooming-flowers-background-frame-loop-video.jpg`) as the animated background.
